# Single Line RPN
One of the shortest RPN calculator in 1 line; 247 characters

## Usage
```shell script
# For infix notation
$ python slinfix.py <expression>
# For postfix notation
$ python slpostfix.py <expression>
```
Example:
```shell script
$ python slinfix.py 2 + 2
4.0
```

## Testing
```shell script
$ python -m unittest
```
