# Single Line RPN
One of the shortest RPN calculator in 1 line; 308 characters

## Usage
```shell script
$ python slrpn.py <expression>
```
Example:
```shell script
$ python slrpn.py 2 2 +
4.0
```

## Testing
```shell script
$ python -m unittest
```
